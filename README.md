# Price Parser Bot

Telegram-бот для мониторинга минимальных цен по поисковым запросам на `Wildberries`, `Ozon` и `Yandex Market`.

Текущая версия использует `Playwright`, чтобы ходить на маркетплейсы как браузер. Это нужно для получения цен ближе к тем, что видны в реальной выдаче, и для частичного обхода антибот-ограничений.

Что умеет бот:

- хранить поисковые запросы пользователя;
- обходить несколько маркетплейсов по одному запросу;
- проходить несколько страниц выдачи на каждой площадке;
- отфильтровывать нерелевантные товары и оставлять только те, где модель действительно совпадает с запросом;
- находить самое дешевое предложение во всей доступной выдаче;
- показывать минимум по каждой выбранной площадке со ссылками или статусом ошибки;
- сохранять историю минимальной цены;
- строить график изменения цены;
- управляться через Telegram-меню и inline-кнопки без ручной работы с внутренними id.

## Что поменялось в логике поиска

Пользовательский запрос теперь является основой поиска. Жесткой зависимости от заранее зашитого списка моделей больше нет: бот использует обычный текст запроса, который ты вводишь сам.

## Важное ограничение

Даже с `Playwright` маркетплейсы могут показывать разные цены в зависимости от региона, авторизации, персонализации, A/B-экспериментов и антибот-эвристик. Поэтому браузерный парсинг заметно лучше простого `httpx`, но он все равно не гарантирует 100% совпадение с тем, что ты видишь в своем обычном браузере.

## Запуск

1. Создай и активируй виртуальное окружение.
2. Установи зависимости:

```bash
pip install -r requirements.txt
```

3. Установи браузер для `Playwright`:

```bash
python -m playwright install chromium
```

4. Скопируй `.env.example` в `.env` и заполни `BOT_TOKEN`.
5. При необходимости настрой глубину пагинации через `MARKETPLACE_MAX_PAGES` и режим через `PLAYWRIGHT_HEADLESS`.
6. Запусти бота:

```bash
python run_bot.py
```

## Основные команды

- `/add cudy wr3000s | wb,ozon,ym` - добавить запрос
- `/list` - открыть список отслеживаний
- `/check 1` - проверить элемент по номеру в списке
- `/chart 1` - показать график по номеру в списке
- `/delete 1` - удалить элемент по номеру в списке
- `/clear` - удалить все отслеживания
- `/menu` - показать главное меню

## Интерфейс в Telegram

После `/start` бот показывает меню с кнопками:

- `Мои отслеживания`
- `Добавить запрос`
- `Проверить все`
- `Удалить все`
- `Помощь`

В списке отслеживаний у каждого элемента есть inline-кнопки:

- `Проверить`
- `График`
- `Удалить`

## Архитектура

- `src/price_parser/main.py` - точка входа
- `src/price_parser/browser.py` - браузерный менеджер Playwright
- `src/price_parser/bot.py` - Telegram-команды, меню и inline-кнопки
- `src/price_parser/db.py` - SQLite-хранилище
- `src/price_parser/monitoring.py` - фоновый мониторинг и уведомления
- `src/price_parser/charting.py` - генерация графиков
- `src/price_parser/query_profiles.py` - нормализация запроса и матчинг моделей
- `src/price_parser/marketplaces/` - адаптеры маркетплейсов
